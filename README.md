# FinanceHub Analytics - Dashboard Executivo de Faturamento

> Plataforma web para análise e monitoramento de faturamento empresarial, desenvolvida com **HTML5**, **CSS3** e **JavaScript ES6+**, utilizando Engenharia de Prompts e Inteligência Artificial para criação de uma aplicação interativa voltada para Business Intelligence e Controladoria.

---

# Sobre o Projeto

O **FinanceHub Analytics** é uma aplicação desenvolvida para transformar dados de faturamento em informações estratégicas para tomada de decisão.

O sistema foi concebido com foco em áreas de **Controladoria**, **FP&A**, **Business Intelligence** e **Gestão Financeira**, oferecendo uma interface executiva para acompanhamento de indicadores, evolução das receitas e análise de desempenho.

Este projeto faz parte de uma iniciativa de desenvolvimento de aplicações financeiras utilizando **Inteligência Artificial Generativa** e **Engenharia de Prompts**, demonstrando como profissionais da área financeira podem construir soluções de alto valor sem depender de frameworks complexos.

---

# Objetivos

- Centralizar indicadores de faturamento em um dashboard executivo.
- Facilitar o acompanhamento da evolução das receitas.
- Apoiar análises gerenciais e estratégicas.
- Automatizar visualizações e indicadores financeiros.
- Demonstrar a aplicação prática de IA no desenvolvimento de soluções para Finanças.
- Compor um portfólio voltado para BI Financeiro, Controladoria e FP&A.

# Engenharia de Prompts

Este projeto foi desenvolvido utilizando um processo estruturado de **Engenharia de Prompts**.

A arquitetura da aplicação, definição dos requisitos funcionais, organização da interface e estrutura analítica foram especificadas por meio de prompts elaborados para Inteligência Artificial Generativa.

A IA foi utilizada como aceleradora do desenvolvimento, enquanto o planejamento da solução, refinamento dos requisitos, validação funcional e evolução do sistema foram conduzidos durante todo o processo de construção.
